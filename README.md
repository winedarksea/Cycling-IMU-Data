# Cycling-IMU-Data
IMU data with classifiers for cycling.

Collected for a LSM6DSV16X IMU's MLC classifier, but should be usable in other IMU environments as well.

Read more about this data here: [https://syllepsis.live/2026/05/10/cycling-imu-data/](https://syllepsis.live/2026/05/10/cycling-imu-data/)

Labels:
```
0 = smooth
1 = rough
2 = gravel
3 = rocky
------
4 = coasting
5 = braking
6 = stationary
7 = pedalling
8 = standing (ie "out of the saddle" riding, climbing)
9 = running
------
10 = straight (not entirely sure on this one, lost the label)
11 = cornering (again, this and 10 might be swapped around, can't remember for sure)
```
